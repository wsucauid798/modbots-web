Mod Bots Web is the Next.js web client for the Mod Bots platform.

## Prerequisites

- Node.js 24 or newer

## Run

Install dependencies:

```powershell
npm install
```

Start the web app:

```powershell
npm run dev
```

The app runs at [http://localhost:3000](http://localhost:3000).
